# Business Sales Performance Analysis Dashboard
## Maincrafts Technology — Business Analytics Internship Task 1

### Project Overview
Analyzed business sales data to identify trends, top products,
and regional performance using Excel and Power BI.

### Tools Used
- Microsoft Excel (Data Cleaning, Pivot Tables, Formulas)
- Power BI Desktop (Dashboard, DAX Measures, Slicers)

### Files in This Repository
| File | Description |
|------|-------------|
| Maincrafts_Sales_Dataset.xlsx | Cleaned dataset with pivot tables |
| Insights_Report.pdf | Key insights and recommendations |
| Process_Documentation.pdf | EDA methodology and approach |
| Dashboard_Screenshot.png | Power BI dashboard preview |

### Key Findings
- Total Sales: Rs. 29,34,235 | Total Profit: Rs. 4,87,106
- Electronics drives 69% of revenue
- South region is top performer at 28.2% share
- Best month: November 2024 at Rs. 4,25,174

### Data Cleaning Highlight
Missing Sales_Amount values were recovered using
Profit Margin-Based Group Imputation — deriving values
mathematically from known Profit and product-level
average margin rates.
